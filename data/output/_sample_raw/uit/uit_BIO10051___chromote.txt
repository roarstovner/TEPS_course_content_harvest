                                                    Innhold
                                        Det legges vekt p at studentene skal f praktisk erfaring med kologiske problemstillinger. Dette inkludert prosessen med innsamling og analyse av data, samt formidling og diskusjoner omkring resultater. Tematisk fokuseres det p evolusjonens betydning for kologien, de fysiske livsbetingelsene, tilgjengelighet av ressurser og gjennomstrmning av stoff og energi, samt inndeling og karakteristikker av jordens biomer. Populasjonskologi, samfunnskologi, biodiversitet, biotiske interaksjoner som konkurranse, predasjon, beiting ogparasitter er andre sentrale tema. Videre berres tema som livshistorie og adferdskologi. I tillegg gis en innfring i viktige miljsprsml relatert til forurensing, bevaring-, forvaltning- og utnyttelse av ressurser.

    function hasScrolled() {
        var st = $(this).scrollTop();

        if (Math.abs(lastScrollTop - st) <= delta)
            return;
        if (st > lastScrollTop && st > navbarHeight) {
            // Scroll ned
            $('nav').removeClass('nav-down').addClass('nav-up');
        } else {
            // Scroll opp
            if (st + $(window).height() < $(document).height()) {
                $('nav').removeClass('nav-up').addClass('nav-down');
            }
        }

        lastScrollTop = st;
